This assignment displays some weather information on seven days. Technologies used are Html5, CSS, JS and Jquery. The api used is from the Aeris weather. In order to use the program one needs to have a working client id and a secret key for Aeris Weather. After entering valid inputs you need to click on "Get Forecast" button. The browser will then request to use the location services. Allow the browser to use the location services. If you block the browser from using the location services, the site will be unable to retrieve data form the api. If the browser does not support using location services then there will be an error message displayed. If everything goes well then the browser will display a table that shows the weather forecast of recent seven days. 

Apart form these, you will have to first download the zipped file unzip it nd open the ss4674_HW2.html file.
